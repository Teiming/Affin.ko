## Start
다음 명령어를 통해 번역된 자료를 다운로드합니다.
```sh
cd ~/Downloads/
git clone https://github.com/Teiming/Affinity-in-Korean.git
sudo ~/Downloads/Affinity-in-Korean/Install.sh
```
